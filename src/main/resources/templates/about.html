Опыт работы с технологиями

В рамках данного проекта использовались следующие технологии:

  • Java - основной язык программирования
  • Spring Boot - фреймворк для создания веб-приложений
  • Spring Data JPA - для работы с базой данных
  • Hibernate - ORM для маппинга объектов на таблицы БД
  • MySQL - система управления базами данных
  • Thymeleaf - шаблонизатор для создания веб-интерфейса
  • Spring Security - для аутентификации и авторизации
  • Maven - система сборки проекта
  • Bootstrap - CSS фреймворк для создания современного интерфейса

Проект позволил получить практический опыт работы с современными технологиями разработки веб-приложений на Java, включая работу с базами данных, реализацию системы безопасности и создание пользовательского интерфейса.
